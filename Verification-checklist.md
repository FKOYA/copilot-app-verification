# GitHub Copilotアプリ 検証チェックリスト

## 運用ルール

- スクリーンショットは「スクショアップロード置き場」Issue(1件)のコメント欄に貼り、発行されたURLを下表の該当行にコピーする
- コメント本文には `{パターン番号}_{分類}_{項目名}` の形式でキーを書いておく(例: `1_basic_mywork`)
- パターンの定義
  - **①**:ローカル(git initなし)
  - **②**:ローカル(git init済み)
  - **③**:リモートリポジトリ接続済み
- 結果列は `✅可` / `❌不可` / `⚠️一部可(備考参照)` / `未検証` のいずれかを記入
