# 検証ログ

GitHub Copilot アプリの検証結果を時系列で記録する生ログ。1 検証 = 「検証手順 → 期待結果 → 実際の結果 → 気づき(特に異常系)」の 4 段構成で追記する。パターンが見えてきた段階で `docs/` に再構成する(詳細は README.md 参照)。

## 記録テンプレート

```markdown
## YYYY-MM-DD: <検証項目名> (Issue #nn)

### 検証手順
1. ...

### 期待結果
- ...

### 実際の結果
- ...

### 気づき(異常系・制約・意外な挙動)
- ...
```
